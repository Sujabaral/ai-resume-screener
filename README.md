# AI Resume Screener & Job Match Scorer

A Flask-based AI project that analyzes how well a resume matches a job description.

## Features
- Upload resume PDF
- Paste job description
- Extract resume text
- Compute job match score
- Highlight matched and missing skills
- Suggest improvements

## Tech Stack
- Python
- Flask
- PyPDF2
- scikit-learn
- Hugging Face Transformers
- HTML/CSS
